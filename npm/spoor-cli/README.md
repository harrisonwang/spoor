# @harrisonwang/spoor-cli

Convert files and URLs to LLM-friendly markdown. See [github.com/harrisonwang/spoor](https://github.com/harrisonwang/spoor) for full docs.

```bash
npm install -g @harrisonwang/spoor-cli
spoor report.pdf
```

Prebuilt binaries are distributed as optional platform packages (`@harrisonwang/spoor-cli-<platform>-<arch>`); npm installs only the one matching your system.
