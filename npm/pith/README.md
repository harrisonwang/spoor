# @harrisonwang/pith

Convert files and URLs to LLM-friendly markdown. See [github.com/harrisonwang/pith](https://github.com/harrisonwang/pith) for full docs.

```bash
npm install -g @harrisonwang/pith
pith report.pdf
```

Prebuilt binaries are distributed as optional platform packages (`@harrisonwang/pith-<platform>-<arch>`); npm installs only the one matching your system.
